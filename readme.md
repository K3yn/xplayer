# xplayer v0.1

A (POC) videoplayer.

Features :
	- Place videos in "sdmc:/media/" folder.
	- U can play mp4 and mkv. 
	- No sound at moment.
	- you can not stop video.
	
Controls :
	- D-pad + A : Menu Selection
	- D-pad + A : Play video
	- Plus : Exit to hbo.
